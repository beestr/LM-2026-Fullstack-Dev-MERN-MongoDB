Flexbox Layout

Flexbox aligns child elements along a row or column. The properties below control direction, spacing, and alignment.

◑

display: flex

Activates Flexbox on the parent container.

↔

justify-content

Distributes space along the main axis (horizontal).

↕

align-items

Aligns items along the cross axis (vertical).
